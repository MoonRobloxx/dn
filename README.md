# HTMLCraft 🎮

**Браузерная строительная игра на HTML5 с оригинальными текстурами**

## 📋 Описание

HTMLCraft — это минималистичная браузерная игра в стиле песочницы, созданная на чистом HTML5, CSS и JavaScript. Рисуйте, строите и разрушайте блоки прямо в браузере!

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| **W/A/S/D** | Движение камеры |
| **↑/←/↓/→** | Альтернативное движение |
| **Q** | Уменьшить масштаб |
| **E** | Увеличить масштаб |
| **ЛКМ** | Поместить блок |
| **ПКМ** | Удалить блок |

## 🎨 Доступные блоки

- 🟫 **Грязь** — основной строительный материал
- 🟢 **Трава** — верхний слой земли
- ⬜ **Камень** — твёрдый материал
- 🟤 **Дерево** — деревянные блоки
- 🟡 **Песок** — в пустынных районах
- 🔵 **Вода** — жидкость
- 🟩 **Листья** — из деревьев
- ✨ **Руда** — ценный ресурс

## 🚀 Запуск

1. Клонируй репозиторий:
```bash
git clone https://github.com/MoonRobloxx/dn.git
cd dn
```

2. Открой файл `index.html` в браузере (или используй Live Server)

3. Наслаждайся игрой! 🎉
